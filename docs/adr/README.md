# Architecture Decision Records (ADR)

이 폴더는 BOOTCAMP_HY 프로젝트의 **아키텍처 결정 기록**을 모아두는 곳입니다.
"왜 이 기술을 골랐는지", "왜 이 구조로 갔는지"를 짧게 남겨서
나중에 봐도 결정의 맥락을 잃지 않도록 합니다.

## Status 종류

- **Proposed** — 도입은 했지만 확정은 아님 (바뀔 가능성 있음)
- **Accepted** — 확정됨, 현재 적용 중
- **Deprecated** — 더 이상 안 쓰지만 옛 코드 호환용으로 남아있음
- **Superseded by ADR-XXXX** — 다른 ADR로 대체됨

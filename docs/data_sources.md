# 데이터 출처 정리 (Data Sources)

> BOOTCAMP_HY 부동산 신뢰 플랫폼이 활용할 4가지 입장별 데이터 출처를 정리한 문서.
> 모든 데이터는 **공공/공식 출처 우선** 원칙을 따른다. (법적 안전성 + 신뢰도)

---

## 0. 데이터 활용 원칙

1. **공공데이터 > 크롤링 데이터 > 사용자 후기** 순으로 신뢰 가중치 부여
2. 크롤링 시 반드시 `robots.txt` 와 사이트 이용약관 확인
3. 개인정보(중개사 실명 포함)는 **공식 공개 정보만** 사용
4. 무료 공식 API 우선 검토 → 없을 때만 다른 방법 고려
5. 데이터 출처는 사용자에게 **항상 명시** (투명성 원칙)

---

## 1. 사용자(임차인/매수인) 관점 데이터

처음 집 구하는 사람이 **"이 거래 안전한가?"** 를 판단할 때 필요한 데이터.

| 데이터 | 출처 | 형태 | 비용 | 우선순위 |
|---|---|---|---|---|
| 아파트/오피스텔 실거래가 | 국토부 실거래가 (rt.molit.go.kr) | 공식 API | 무료 | ★★★ |
| 단지 실거주 리뷰 | 호갱노노 (hogangnono.com) | 크롤링 (약관 확인 필요) | 무료 | ★★ |
| 신축 단지 리뷰 | 디스코 (disco.re) | 크롤링 (약관 확인 필요) | 무료 | ★ |
| 전세사기 통계 | HUG 보증사고 통계 (khug.or.kr) | 보도자료/PDF | 무료 | ★★ |
| Q&A 데이터(학습용) | 부동산스터디 네이버 카페 | 회원 가입 후 열람 | 무료 | ★★ (AI 학습용) |
| 매물 리뷰 | 직방/다방 | 제한적 | - | ★ |

### 활용 방향
- 실거래가 API → **시세 적정성 자동 판단** (호가 vs 실거래)
- HUG 통계 → **위험 지역 가중치**
- 카페 Q&A → **AI 감정분석 학습 데이터** (※ 크롤링 약관 반드시 확인)

## 2. 부동산협회(KAR) 관점 데이터

공인중개사 자격과 윤리의 **공식 검증** 출처.

| 데이터 | 출처 | 형태 | 비용 | 우선순위 |
|---|---|---|---|---|
| 공인중개사 자격 조회 | 한국공인중개사협회 (kar.or.kr) | 웹 검색 (API 없음) | 무료 | ★★★ |
| 회원 사무소 검색 | 협회 회원 검색 | 웹 검색 | 무료 | ★★ |
| 자격 정지/취소 이력 | 협회 공시 | 공시 페이지 | 무료 (제한적) | ★★ |
| 위반 통계 | 협회 보도자료 | PDF/HTML | 무료 | ★ |

### 활용 방향
- 자격 유효 여부 → **신뢰 점수 +10점** (Section 4 점수 공식 참고)
- 협회 정회원 여부 → **신뢰 점수 +5점**
- 자격 정지 이력 → **감점 -15점**

### 주의사항
- 협회는 공식 API를 제공하지 않음 → **웹 스크래핑 시 협회 동의 확인 권장**
- 가능하면 **협회 협력/제휴 형태로 정식 데이터 제공받는 게 베스트**

## 3. 중개사 사업체 관점 데이터

중개사가 **합법적이고 안정적으로 운영 중인지** 확인하는 데이터.

| 데이터 | 출처 | 형태 | 비용 | 우선순위 |
|---|---|---|---|---|
| 사업자등록 상태 | 국세청 (api.odcloud.kr) | 공식 API | 무료 | ★★★ |
| 부동산중개업 등록 | 지자체별 (예: 서울부동산정보광장 land.seoul.go.kr) | 지역마다 다름 | 무료 | ★★★ |
| 개업/폐업 이력 | 국세청 + 지자체 | 부분 공개 | 무료 | ★★ |
| 매물 광고 수 | 네이버부동산/직방 | 크롤링 필요 | - | ★ |
| 거래 건수 (추정) | 실거래가 데이터에서 역추적 | 가공 필요 | 무료 | ★ |

### 활용 방향
- 사업자등록 정상 → **신뢰 점수 +10점** (즉시 확인 가능, 자동화 쉬움)
- 부동산중개업 등록증 보유 → **15점** (사용자 기존 안 유지)
- 10년 이상 운영 → **15점** (사용자 기존 안 유지)

### 우선 구현 추천
국세청 사업자등록상태 API는 **무료 + 즉시 신뢰도 +10점 효과** 라서 가장 먼저 붙이는 게 가성비 최고.

## 4. 국가/공공 관점 데이터 (가장 신뢰성 높음)

공공기관이 직접 관리하는 **공식 데이터**. 법적 안전성 + 정확성 모두 최상.

| 데이터 | 출처 | API | 우선순위 |
|---|---|---|---|
| 부동산 실거래가 | rt.molit.go.kr (국토부) | ★ 공식 API | ★★★ |
| 공간정보 (지도/지번) | V-World (vworld.kr) | ★ 공식 API | ★★★ |
| 건축물대장 + 위반건축물 | 세움터 (eais.go.kr) | API 있음 | ★★ |
| 부동산종합공부 | 일사편리 (kras.go.kr) | 일부 API | ★★ |
| 토지이용계획 | LURIS (luris.molit.go.kr) | API 있음 | ★ |
| 주택 통계 | KOSIS (kosis.kr) | API 있음 | ★ |
| 전세보증사고 통계 | HUG (khug.or.kr) | 통계만 공개 | ★★ |
| 안심전세 시세 | HUG 안심전세 앱 | 비공식 | ★ |
| 사업자등록상태 | 국세청 (api.odcloud.kr) | ★ 공식 API | ★★★ |
| 등기부등본 | 인터넷등기소 (iros.go.kr) | API 없음, 유료 | ★ (자동화 어려움) |

### 활용 방향
- 실거래가 → **시세 적정성 자동 판단**
- 건축물대장 → **위반 건축물 경고** (사용자 알림)
- HUG 사고이력 → **위험 신호 -30점** (강제 70점 미만으로 강등)
- 국세청 사업자 → **신뢰 점수 +10점**

### 등기부등본 관련 주의
등기부는 **공인된 위험 판단 자료**지만 API가 없고 유료(통당 700원).
→ **자동화는 어렵고, 사용자가 직접 발급한 등기부를 업로드하면 OCR로 분석** 하는 방향이 현실적.

## 5. 데이터 우선 구현 순서 (추천)

법적 안전성 + 가성비 + 즉시 효과 기준.

```
1단계 (1~2주 이내, 가성비 최고)
  ├─ 국세청 사업자등록상태 API 연동
  ├─ 국토부 실거래가 API 연동
  └─ V-World 지도/지번 API 연동

2단계 (1개월 이내)
  ├─ 세움터 건축물대장 API 연동 (위반 건축물 확인)
  ├─ HUG 전세사고 통계 수동 매핑 (지역별)
  └─ 한국공인중개사협회 자격 조회 (수동 검증 우선)

3단계 (사용자 후기 데이터 쌓인 후)
  ├─ 익명 게시판 AI 감정분석
  ├─ 실명 리뷰 점수화
  └─ 베이지안 평균으로 신규 중개사 차별 방지

4단계 (장기, 협력 필요)
  ├─ 한국공인중개사협회 공식 제휴 (자격 데이터 정식 수신)
  ├─ 호갱노노/디스코 등 외부 리뷰 정식 제휴
  └─ 등기부 OCR 업로드 분석
```

## 6. 법적 리스크 체크리스트

데이터를 다룰 때 반드시 점검할 사항.

- [ ] 크롤링 대상 사이트의 `robots.txt` 와 이용약관 확인
- [ ] 개인정보(이름/연락처/주소)는 **공식 공개 정보만** 수집
- [ ] 중개사 부정 평점 공개 시 **사실 기반 + 익명 작성자 보호**
- [ ] 후기 작성자 신원은 **해시 처리 후 저장**
- [ ] "최고의 중개사" 같은 **주관적 단정 표현 금지** (표시광고법)
- [ ] 점수 산정 공식은 사이트에 **공개**
- [ ] 변호사 자문 **최소 1회** (대한법률구조공단 무료 상담 가능)
